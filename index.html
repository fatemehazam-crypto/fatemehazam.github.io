In-Progress Work: Rust-Based Secure Commander Middleware for Drone Fleets

Building a high-performance Rust security gateway that prevents command-injection and hijacking attacks in autonomous drone fleets deployed in smart-city environments.

Overview

In modern smart-city operations, drones manage real-time traffic monitoring, inspections, and emergency response. However, these systems are vulnerable: attackers can inject malicious flight commands through radio links, causing route deviations, stalling missions, or forcing unsafe maneuvers.
